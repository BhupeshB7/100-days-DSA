# 💯 100 Days of DSA Challenge

Welcome to my **100 Days of DSA** (Data Structures & Algorithms) challenge!  
This repository contains my daily solutions to the most popular DSA problems categorized by topic — ranging from arrays and strings to graphs and backtracking.

---

## 📅 Challenge Overview

- ⏳ **Duration:** 100 Days
- 📌 **Goal:** Solve handpicked DSA problems daily to strengthen my problem-solving skills.
- 🧠 **Focus Areas:** Arrays, Strings, Linked Lists, Trees, Graphs, DP, Recursion, Sliding Window, and more.
- ✏️ **Approach:** Solve, Understand, and Document — Every problem includes notes/approaches or links to external explanations.
